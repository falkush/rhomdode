# Controls
w: go foward

a: go left

s: go backward

d: go right

z: increase red value

x: increase green value

c: increase blue value

v: decrease block distance

b: increase block distance

n: toggle construction/destruction mode

m: toggle fast/slow speed

p: reset

# Requirements
- Windows
- Nvidia GPU
- 64-bits architecture
